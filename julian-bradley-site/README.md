# Julian Bradley Website

A modern, responsive website for Julian Bradley featuring a silver, maroon, and black color scheme.

## Features

- **Modern Design**: Clean, professional layout with silver, maroon, and black color scheme
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling
- **Mobile-Friendly**: Mobile menu toggle for easy navigation on small screens
- **Optimized Images**: Images downloaded from the original site and optimized

## Color Scheme

- **Silver**: #C0C0C0 (primary text and accents)
- **Maroon**: #800020 (accent color for highlights and CTAs)
- **Black**: #000000 (background and primary elements)

## File Structure

```text
julian-bradley-site/
```

## Usage

1. Open `index.html` in a web browser
2. Or serve using a local web server:

   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```
