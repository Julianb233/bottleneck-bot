# Pain Point Emotional Images

This directory contains emotional/distress images that are displayed when users click on pain point cards in the landing page.

## Required Images

Please add the following images to this directory:

1. **ghl-chaos.jpg** - Image showing the stress/chaos of managing multiple GHL sub-accounts
   - Suggested themes: Overwhelmed person at computer, multiple browser tabs, chaos, stress
   
2. **meta-ads-fire.jpg** - Image showing the distress of Meta Ads burning money
   - Suggested themes: Person waking up stressed, money burning, dashboard showing losses, 3am phone check
   
3. **leads-bump.jpg** - Image showing missed opportunities and leads falling through
   - Suggested themes: Missed phone calls, empty calendar slots, frustrated person, lost opportunities
   
4. **missed-calls.jpg** - Image showing lost revenue from missed calls
   - Suggested themes: Phone with missed calls, money walking away, frustrated business owner, lost deals
   
5. **reporting-nightmare.jpg** - Image showing the exhaustion of manual reporting
   - Suggested themes: Person working late at night, multiple spreadsheets, exhaustion, midnight work
   
6. **zapier-breaks.jpg** - Image showing the chaos when automations break
   - Suggested themes: Broken chain, system failure, panic, damage control mode

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Recommended 1200x800px or larger (will be scaled responsively)
- **Emotional Tone**: Should invoke empathy and relate to agency owner struggles
- **Quality**: High resolution, professional quality
- **Content**: Should be relatable to agency owners dealing with these specific pain points

## Usage

These images are automatically loaded when users click on the corresponding pain point cards in the landing page. The modal will display a fallback message if an image is missing.
